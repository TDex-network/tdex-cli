# alpha-wallet
Wallet CLI for making swaps on TDEX

## Development

**Requirements**

* Node/npm or yarn


* Install deps

```sh
yarn install
```

* Build .ts files

```sh
yarn build
```

* Build and Test

```sh
yarn test
``` 

* Try it out locally

```sh
npm install -g 
tdex-cli --version
```


## Usage

* Install

```sh
npm install -g tdex-cli
```

* Connect to a liquidity provider

```sh
tdex-cli connect https://tdex.vulpem.com
```

* Select a market to use for trading

```sh
tdex-cli market LBTC-USDT
```
