Non-custodial Liquid Wallet CLI to make trades on TDex Network via atomic swaps 


## Development

### Requirements

* Node/npm or yarn

### Instructions

* Install deps

```sh
yarn install
```

* Build .ts files

```sh
yarn build
```

* Build and Test

```sh
yarn test
``` 

* Try it out locally

```sh
npm install -g 
tdex-cli --version
```
